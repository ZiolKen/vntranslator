Get Your DeepSeek API Key

  1. Visit the official DeepSeek website:
    https://platform.deepseek.com/
  2. If you do not have an account, click Sign Up and register with your email address.
    If you already have an account, click Log In.
  3. After logging in, go to your API Keys page (sidebar menu or via profile/avatar).
  4. Click Create API Key, give it any name, and confirm.
    Copy your new API Key and keep it safe. You will need it for this tool.
  5. Top up your balance (if required) to use DeepSeek services (Billing/Top Up menu).

If you have any issues, visit the DeepSeek support page.

Note: Never share your API key. If you think your key is compromised, delete it and create a new one.
Translation not working?
If your translation fails, it usually means your DeepSeek balance is insufficient. Please top up your account on DeepSeek and try again.
