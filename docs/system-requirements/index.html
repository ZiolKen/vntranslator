System requirements

VN Translator runs in the browser. For the smoothest experience, use a modern desktop browser.

Recommended setup

  • OS: Windows / macOS / Linux
  • Browser: latest Chrome / Edge / Firefox
  • RAM: 8 GB+ recommended for large projects
  • Storage: enough space for exported files + backups

Mobile support

You can open the docs and some tools on mobile, but serious translation work is best on desktop because:

  • large JSON and .rpy files require memory
  • long sessions are easier with a keyboard
  • some features (like folder access) vary by browser

Browser features used

Some tools use modern APIs:

  • File picker: for uploading a file
  • IndexedDB: for stable local session storage (translation state, preview sync, auto-save)
  • File System Access API (optional): for folder-based workflows (best on Chromium browsers)

If a feature is not supported in your browser:

  • use a file-based workflow (export/import ZIP or single files)
  • or switch to Chrome/Edge for advanced folder access

Connectivity

  • Translation APIs require a stable internet connection.
  • If you use a free service (Lingva/Google), expect occasional downtime or throttling.

Troubleshooting performance

If the UI becomes slow:

  • translate fewer files at a time
  • lower batch size
  • close other heavy tabs
  • clear old sessions (see: Privacy & security)
