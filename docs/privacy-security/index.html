Privacy & security

VN Translator is designed for a local-first workflow, but translation services are online by nature.

API keys are secrets

If you use DeepSeek / OpenAI / DeepL:

  • Treat your API key like a password.
  • Never share it in screenshots, recordings, or public repos.
  • Prefer creating a dedicated key for VN Translator.

See: API keys

What is stored locally

Depending on the tool, VN Translator may store:

  • translation progress
  • preview/editor state
  • recent files metadata (not the original file itself)
  • your settings (target language, batch size)

Storage is kept in the browser (commonly IndexedDB).

How to clear it

  • Open browser settings → Site data / Storage
  • Find vntranslator.pages.dev
  • Clear data

What is sent over the network

When you start translation:

  • Only the text segments needed for translation are sent to the provider you choose.
  • The provider may log requests according to their policies.

Legal / ethical note

Only translate and modify content that you own or are allowed to modify.
VN Translator is a tooling platform — use it responsibly.
