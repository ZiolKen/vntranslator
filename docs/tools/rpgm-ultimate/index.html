RPGM Ultimate

RPGM Ultimate is designed for multi-file RPG Maker workflows.

Open the tool: RPGM Ultimate

What it does

  • Step 1: Load one or more JSON files → export a TXT package for translation
  • Step 2: Translate the TXT (DeepSeek/DeepL/Lingva/Google depending on UI)
  • Import translated TXT back → export final JSON files

This reduces risk when translating many files because you can:

  • translate in a consistent batch format
  • re-run translation on unfinished lines
  • keep JSON structure stable

Step 1 — Extraction & packaging

  1. Select original JSON file(s) from your data/ folder
  2. Click Export TXT
  3. You now have a translation-friendly TXT

Step 2 — Auto translation

  1. Select the TXT file exported from Step 1
  2. Choose target language + translation service
  3. Set batch size (start small)
  4. Translate
  5. Import the translated TXT back into Step 1
  6. Save/export JSON

Best practices

  • Keep your original data/ folder backed up
  • Translate one file first (e.g., System.json) to validate your workflow
  • Use Preview/editor steps in the tool to check control codes

If you only need a single file quickly:

  • use RPG Maker Translator
