Ren’Py Compiler

Open the tool: Ren’Py Compiler

This tool provides utilities often used in Ren’Py development pipelines.

What it can do (high level)

  • Extract files from .rpa / .rpi archives
  • Decompile .rpyc back to readable scripts (when possible)
  • Repack archives

Important note about permissions

Only use this tool on:

  • your own projects, or
  • projects where you have explicit permission to modify assets/scripts

VN Translator does not encourage bypassing protections or violating licenses.

Recommended use cases

  • Recovering scripts from your own builds
  • Debugging your packaging pipeline
  • Preparing files for legitimate localization work

If you are translating a game you did not develop:

  • ask the developer for source files instead
