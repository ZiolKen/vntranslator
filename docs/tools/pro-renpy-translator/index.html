Pro Ren’Py Translator

This is the “project workflow” tool for Ren’Py: multi-file editing, translation memory, and export options designed for long-term work.

Open the tool: Pro Ren’Py Translator

Key features

  • Open multiple files or a whole folder
  • Translate only missing/untranslated strings
  • Translation Memory (TM) to reuse previous work
  • Find/replace, undo/redo, copy source/translation
  • Export a single file or export the entire project as ZIP
  • Placeholder warnings to prevent broken scripts

Recommended workflow

  1. Open Pro Ren’Py Translator
  2. Click Open Folder and select your Ren’Py project folder (best)
    Or use Open Files to select specific scripts.
  3. Choose engine (DeepSeek / DeepL / Lingva) and target language
  4. Enable:
    • Fill missing from TM before MT (recommended)
    • Auto-save (recommended)
  5. Click Translate Untranslated
  6. Review warnings and fix issues manually
  7. Export:
    • Export File for a single script
    • Export ZIP for the whole translated set
  8. Test in Ren’Py

Translation Memory tips

  • Use TM when you iterate on the same project
  • It improves consistency for:
    • character names
    • UI terms
    • recurring phrases
  • TM also reduces cost (less MT calls)

Safe settings

If you are unsure:

  • Extract mode: Safe
  • Batch size: 50 or lower
  • Keep placeholder warnings enabled

When something looks wrong

  • filter by Error / Flag
  • use search to find broken tags ({/i} missing, etc.)
  • fix, then export again
