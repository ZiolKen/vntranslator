Game Text Editor

The Game Text Editor is for manual editing and QA across common VN/RPG formats.

Open the tool: Game Text Editor

Use cases

  • Fix a few lines after machine translation
  • Search/replace character names
  • Normalize punctuation or spacing
  • Quickly check a file without opening an IDE

Recommended workflow

  1. Open Game Text Editor
  2. Load a file (.rpy, .json, .txt, … depending on tool support)
  3. Use find/replace for consistent terminology
  4. Save/export and test in game

Safety tips

  • Never remove engine control codes / placeholders
  • For JSON, make sure escapes stay valid
  • Keep backups before large replaces
