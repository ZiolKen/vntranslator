RPG Maker Translator

Use this tool to translate a single RPG Maker MV/MZ JSON file (from data/).

Open the tool: RPG Maker Translator

Inputs

  • .json files from YOUR_GAME/data/
    • System.json, Actors.json, Items.json, CommonEvents.json, Map001.json, …

Guide: How to get RPGM JSON

Step-by-step

  1. Open RPG Maker Translator
  2. Select engine/model (DeepSeek / OpenAI / Lingva, depending on tool UI)
  3. Paste your API key if required
  4. Upload one .json
  5. Translate (start with small batch size)
  6. QA in Preview & Editor
  7. Export the translated .json
  8. Replace the file back into your project’s data/ folder
  9. Playtest in RPG Maker

Control codes you must preserve

RPG Maker uses control codes in message strings. Do not translate or break them.

Code Meaning
\V[1] Variable value
\N[3] Actor name
\P[1] Party member name
\C[2] Text color
\I[10] Icon
\G Currency unit
\{ / \} Text size up/down

Also keep:

  • JSON escapes like \\n
  • quotes and commas
  • line breaks that affect message windows

Preview & Editor

Use Preview to verify:

  • control codes unchanged
  • \ still correctly escaped in JSON
  • line breaks look natural in the message window

If you see Invalid JSON after export:

  • a quote or backslash was broken
  • re-open in editor, fix escapes, export again

When to use RPGM Ultimate instead

If you have many files to translate:

  • use RPGM Ultimate (JSON → TXT pipeline)
