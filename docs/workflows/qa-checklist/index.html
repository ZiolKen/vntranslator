QA checklist

Use this as a final pass before you ship a translated build.

Placeholder safety

  • [ ] Ren’Py tags are balanced ({i} has {/i})
  • [ ] Ren’Py variables unchanged ([name])
  • [ ] RPG Maker control codes preserved (\V[1], \N[3], \C[2])
  • [ ] JSON escaping valid (\\ where required)

Language quality

  • [ ] consistent terminology (names, places, UI words)
  • [ ] punctuation and spacing look natural
  • [ ] line breaks fit message windows

UI & layout

  • [ ] menus aren’t overflowing
  • [ ] choice buttons are readable
  • [ ] long strings are wrapped or shortened

Testing

  • [ ] open the game and click through key scenes
  • [ ] test saves/loads in multiple places
  • [ ] test shops, items, system menus (RPG Maker)
  • [ ] test screens, settings, quick menu (Ren’Py)

If anything breaks

  • fix the specific line in the editor
  • export again
  • retest immediately

QA is an iterative loop — don’t wait until the end to test.
