FAQ

Do I need to install anything?

No. VN Translator is web-based.

Does VN Translator upload my files?

Files are processed locally in your browser, but translation requires sending text segments to the chosen translation service.

Which translation service is best?

For most VN workflows, start with DeepSeek.
For maximum nuance, OpenAI can be stronger.
Always QA formatting.

See: Translation services

Where are my sessions saved?

Most tools store progress locally using browser storage (often IndexedDB). You can clear it in browser settings.

Can I translate a game I downloaded?

Only if you own the rights or have permission. The safest workflow is to ask the developer for source files.

My exported file breaks the game. What now?

Check placeholders:

  • Ren’Py tags and [variables]
  • RPG Maker control codes and JSON escaping
